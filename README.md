MyResume Bootstrap Theme For XOOPS CMS 2.5.8.x
=====

![Theme Preview](/theme_preview.png)

My Resume is a clean, multipurpose and well crafted theme. It’s a cv / resume / vCard focused on digital professionals, whether designers, programmers or photographers. It is perfect to promote your work. My Resume built with responsive Bootstrap 3.3.

Ported to XOOPS [CMS 2.5.8.x](http://xoops.org/modules/news/article.php?storyid=6762)
